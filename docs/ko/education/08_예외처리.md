# 예외 처리
> 오류를 처리하는 세 가지 방법: `try/catch` (예외 기반), `Option<T>` (값 부재), `Result<T>` (성공/실패 + 오류 정보)

---

## try / catch

```
try {
    본문
} catch (변수명) {
    오류 처리
};
```

```dri
try {
    int result = 10 / 0;
    print(result);
} catch (err) {
    print("오류:", err);
};
```

---

## throw

```
throw 표현식;
```

`throw`는 String 메시지를 던집니다.

```dri
double divide(double a, double b) {
    if (b == 0.0) {
        throw "0으로 나눌 수 없습니다";
    };
    return a / b;
};

try {
    double r = divide(10.0, 0.0);
    print(r);
} catch (err) {
    print("예외:", err);
};
# 출력: 예외: 0으로 나눌 수 없습니다
```

## 중첩 예외 처리

```dri
void process_file(String path) {
    if (length(path) == 0) {
        throw "경로가 비어 있습니다";
    };
    String content = io.read_file(path);
    if (length(content) == 0) {
        throw "파일이 비어 있습니다";
    };
    print("처리 완료:", length(content), "바이트");
};

try {
    process_file("");
} catch (err) {
    print("실패:", err);   # 실패: 경로가 비어 있습니다
};

try {
    process_file("data.txt");
} catch (err) {
    print("실패:", err);
};
```

## Option\<T\> — 값이 있거나 없거나

값이 없을 수 있는 조회 연산에 사용합니다. `Some(값)` 또는 `None`을 반환합니다.

```dri
Option<int> safe_div(int a, int b) {
    if (b == 0) {
        return None;
    };
    return Some(a / b);
};

match safe_div(10, 2) {
    Some(v) => { print("결과:", v); };   # 결과: 5
    None    => { print("실패"); };
};

match safe_div(10, 0) {
    Some(v) => { print("결과:", v); };
    None    => { print("실패"); };       # 실패
};
```

### 검색에 Option 활용

```dri
Option<int> find_first_even(int[] arr) {
    for (n of arr) {
        if (n % 2 == 0) {
            return Some(n);
        };
    };
    return None;
};

int[] data = [1, 3, 5, 4, 7];
match find_first_even(data) {
    Some(v) => { print("첫 짝수:", v); };   # 첫 짝수: 4
    None    => { print("짝수 없음"); };
};
```

## Result\<T\> — 성공 또는 실패 (오류 정보 포함)

실패 이유를 전달해야 하는 변환·검증 연산에 사용합니다. `Ok(값)` 또는 `Err(메시지)`를 반환합니다.

```dri
Result<int> parse_int(String s) {
    if (length(s) == 0) {
        return Err("빈 문자열");
    };
    return Ok(str.to_int(s));
};

match parse_int("42") {
    Ok(v)  => { print("성공:", v); };    # 성공: 42
    Err(e) => { print("오류:", e); };
};

match parse_int("") {
    Ok(v)  => { print("성공:", v); };
    Err(e) => { print("오류:", e); };   # 오류: 빈 문자열
};
```

### 연쇄 Result 처리

```dri
Result<double> safe_sqrt(double x) {
    if (x < 0.0) {
        return Err("음수의 제곱근은 실수가 아닙니다");
    };
    return Ok(sqrt(x));
};

Result<double> compute(String input) {
    match parse_int(input) {
        Ok(n) => {
            match safe_sqrt(n as double) {
                Ok(v)  => { return Ok(v); };
                Err(e) => { return Err(e); };
            };
        };
        Err(e) => { return Err("파싱 오류: " + e); };
    };
};

match compute("25") {
    Ok(v)  => { print("결과:", v); };   # 결과: 5.0
    Err(e) => { print("오류:", e); };
};

match compute("-4") {
    Ok(v)  => { print("결과:", v); };
    Err(e) => { print("오류:", e); };   # 오류: 음수의 제곱근은 실수가 아닙니다
};
```

## try + Option/Result 혼합

```dri
void run_job(String filename) {
    try {
        String content = io.read_file(filename);
        match parse_int(content) {
            Ok(n)  => { print("파일 값:", n * 2); };
            Err(e) => { throw "파싱 실패: " + e; };
        };
    } catch (err) {
        print("작업 실패:", err);
    };
};

run_job("number.txt");
run_job("missing.txt");
```

## 오류 처리 전략 비교

| 방식 | 사용 상황 |
|------|----------|
| `try/catch` | 외부 자원(파일, 네트워크) 접근 |
| `Option<T>` | 값이 없을 수 있는 조회 연산 |
| `Result<T>` | 실패 이유를 전달해야 하는 변환·검증 |

> `match` 패턴 매칭 문법은 [[04_제어문.md](04_제어문.md)]를 참고하세요.
