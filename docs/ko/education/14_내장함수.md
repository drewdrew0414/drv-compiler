# 내장 함수 레퍼런스
> drv에서 기본 제공하는 내장 함수와 네임스페이스별 함수 목록이다.

---

## 네임스페이스 규칙

drv 함수는 세 가지 계층으로 구분합니다.

| 계층 | 예시 | 판별 기준 |
|------|------|----------|
| **글로벌** | `print`, `abs`, `max`, `sqrt`, `length` | 거의 모든 프로그램에 등장하는 핵심 연산 |
| **네임스페이스** | `math.pow`, `lst.sort`, `str.join`, `io.read_file` | 도메인별 확장 함수 |
| **tensor 전용 글로벌** | `sum`, `dot`, `norm`, `mean` | tensor 타입에만 동작 — 타입 시스템이 충돌 방지 |

> 판별 기준: **"거의 모든 drv 파일에 쓰이는가?"** → YES면 글로벌, NO면 네임스페이스.

```drv
# 글로벌 — 네임스페이스 없이 직접 호출
abs(-5)         # 5
max(3, 7)       # 7
sqrt(16.0)      # 4.0
length("hello") # 5

# 네임스페이스 — 반드시 접두사 필요
math.pow(2.0, 10.0)     # 1024.0
lst.sort(fruits)         # 정렬된 복사본
str.join(words, ", ")    # 문자열 결합
io.read_file("a.txt")    # 파일 읽기
```

## 기본 I/O

| 함수 | 시그니처 | 설명 |
|------|----------|------|
| `print` | `print(...)` | 값 출력 (공백 구분, 개행 포함) |

```drv

String s;

```

## 수학 함수 (직접 호출)

| 함수 | 인수 | 설명 |
|------|------|------|
| `abs(x)` | 1 | 절댓값 |
| `max(a, b)` | 2 | 최댓값 |
| `min(a, b)` | 2 | 최솟값 |
| `sqrt(x)` | 1 | 제곱근 |
| `floor(x)` | 1 | 내림 |
| `ceil(x)` | 1 | 올림 |
| `round(x)` | 1 | 반올림 |
| `sin(x)` | 1 | 사인 (라디안) |
| `cos(x)` | 1 | 코사인 (라디안) |
| `tan(x)` | 1 | 탄젠트 (라디안) |
| `fma(a, b, c)` | 3 | a\*b+c (Fused Multiply-Add) |
| `likely(x)` | 1 | 분기 예측 힌트: 참일 가능성 높음 |
| `unlikely(x)` | 1 | 분기 예측 힌트: 거짓일 가능성 높음 |

```drv
print(abs(-5));              # 5
print(max(3, 7));            # 7
print(min(3, 7));            # 3
print(sqrt(16.0));           # 4.0
print(floor(3.7));           # 3.0
print(ceil(3.2));            # 4.0
print(round(3.5));           # 4.0
print(sin(3.14159 / 2.0));   # ~1.0
print(fma(3.0, 4.0, 5.0));   # 17.0
```

## 컬렉션 공통

| 함수 | 인수 | 설명 |
|------|------|------|
| `length(x)` | 1 | 배열/문자열 길이 |
| `split(s, delim)` | 2 | 문자열 분할 → `list<String>` |

```drv
int[] arr = [1, 2, 3];
print(length(arr));         # 3
print(length("hello"));     # 5

list<String> parts = split("a,b,c", ",");
for (p of parts) { print(p); };
# a, b, c
```

## 문자열 변환 내장 함수

> `toUpper`, `toLower`, `trim`, `replace`는 `str.*`가 아닌 **직접 호출** 함수입니다.

| 함수 | 인수 | 설명 |
|------|------|------|
| `toUpper(s)` | 1 | 대문자 변환 |
| `toLower(s)` | 1 | 소문자 변환 |
| `trim(s)` | 1 | 앞뒤 공백 제거 |
| `replace(s, old, new)` | 3 | 문자열 치환 |

```drv
print(toUpper("hello"));                      # HELLO
print(toLower("WORLD"));                      # world
print(trim("  hi  "));                        # "hi"
print(replace("foo bar", "bar", "baz"));      # foo baz
```

## str.* 네임스페이스

| 함수 | 인수 | 설명 |
|------|------|------|
| `str.char_at(s, i)` | 2 | i번째 문자 |
| `str.from_char(c)` | 1 | char → String |
| `str.from_int(n)` | 1 | int → String |
| `str.to_int(s)` | 1 | String → int |
| `str.substr(s, start, len)` | 3 | 부분 문자열 |
| `str.starts_with(s, prefix)` | 2 | 접두사 확인 |
| `str.ends_with(s, suffix)` | 2 | 접미사 확인 |
| `str.contains(s, sub)` | 2 | 포함 여부 |
| `str.repeat(s, n)` | 2 | n번 반복 |
| `str.simd_find(s, t)` | 2 | SIMD 부분 문자열 검색 (인덱스, 없으면 -1) |

```drv
print(str.char_at("hello", 1));               # 'e'
print(str.from_char('A'));                    # "A"
print(str.from_int(42));                      # "42"
print(str.to_int("99"));                      # 99
print(str.substr("hello world", 6, 5));       # "world"
print(str.starts_with("drv lang", "drv"));    # true
print(str.ends_with("file.drv", ".drv"));     # true
print(str.contains("hello world", "world"));  # true
print(str.repeat("ha", 3));                   # "hahaha"
print(str.simd_find("hello world", "world")); # 6
```

## lst.* 네임스페이스 (list 연산)

> `list.*`가 아닌 **`lst.*`** 입니다.

| 함수 | 인수 | 설명 |
|------|------|------|
| `lst.push(l, val)` | 2 | 끝에 추가 |
| `lst.pop(l)` | 1 | 끝 요소 제거 후 반환 |
| `lst.size(l)` | 1 | 크기 (long) |
| `lst.get(l, i)` | 2 | i번째 요소 |
| `lst.set(l, i, val)` | 3 | i번째 요소 수정 |

```drv
list<int> nums = [];
lst.push(nums, 10);
lst.push(nums, 20);
lst.push(nums, 30);
print(lst.size(nums));       # 3
print(lst.get(nums, 1));     # 20
lst.set(nums, 1, 99);
int v = lst.pop(nums);       # 30
print(lst.size(nums));       # 2
```

## map.* 네임스페이스

| 함수 | 인수 | 설명 |
|------|------|------|
| `map.set(m, key, val)` | 3 | 삽입/수정 |
| `map.get(m, key)` | 2 | 값 조회 |
| `map.get_or(m, key, default)` | 3 | 기본값 조회 |
| `map.has(m, key)` | 2 | 키 존재 여부 |
| `map.del(m, key)` | 2 | 키 삭제 |
| `map.keys(m)` | 1 | 모든 키 목록 |
| `map.size(m)` | 1 | 항목 수 (long) |

```drv
Map<String, int> m = {};
map.set(m, "a", 1);
map.set(m, "b", 2);
print(map.get(m, "a"));          # 1
print(map.get_or(m, "c", 0));   # 0
print(map.has(m, "b"));          # true
map.del(m, "b");
print(map.size(m));              # 1
```

## tensor / 통계 함수 (직접 호출)

| 함수 | 인수 | 설명 |
|------|------|------|
| `sum(t)` | 1 | 원소 합 (AVX 자동 디스패치) |
| `dot(a, b)` | 2 | 내적 |
| `norm(t)` | 1 | 유클리드 노름 |
| `mean(t)` | 1 | 평균 |
| `std(t)` | 1 | 표준편차 |
| `median(t)` | 1 | 중앙값 |
| `min_index(t)` | 1 | 최솟값 인덱스 |
| `max_index(t)` | 1 | 최댓값 인덱스 |
| `avx512_sum(t)` | 1 | AVX-512 직접 합 |
| `avx512_dot(a, b)` | 2 | AVX-512 직접 내적 |
| `fast_csv_read(path)` | 1 | 고속 CSV 파싱 |

```drv
tensor<5, double> data = {3.0, 1.0, 4.0, 1.0, 5.0};
print(sum(data));          # 14.0
print(mean(data));         # 2.8
print(norm(data));         # ~5.83
print(min_index(data));    # 1
print(max_index(data));    # 4
```

## io.* 네임스페이스

| 함수 | 인수 | 설명 |
|------|------|------|
| `io.read_file(path)` | 1 | 파일 읽기 → String |
| `io.write_file(path, content)` | 2 | 파일 쓰기 (덮어쓰기) → boolean |
| `io.append_file(path, content)` | 2 | 파일 추가 쓰기 → boolean |
| `io.mmap_read(path)` | 1 | 메모리 맵 파일 읽기 (복사 없이 직접 접근) |

```drv
io.write_file("out.txt", "Hello, drv!");
String content = io.read_file("out.txt");
print(content);   # Hello, drv!
io.append_file("out.txt", "\n두 번째 줄");
```

> **`io.mmap_read` 사용 시 주의**: 반환된 매핑 참조를 `@region` 블록 밖으로 탈출시키면  
> **Use-After-Free** 취약점이 발생합니다. 컴파일러의 탈출 분석이 이를 컴파일 오류로 차단합니다.  
> 자세한 내용은 [[10_메모리모델.md — @region 탈출 분석](10_메모리모델.md)]을 참고하세요.
