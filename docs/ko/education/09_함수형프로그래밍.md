# 함수형 프로그래밍
> 람다 표현식, 고차 함수(map/filter/reduce), 파이프 연산자로 데이터 변환 파이프라인을 구성한다.

---

## 람다 표현식

```
|매개변수, ...| -> 표현식
|매개변수, ...| { 본문 }
```

```drv
|x| -> x * 2
|x, y| -> x + y
|x| -> x > 0
|| -> 42             # 매개변수 없음
```

블록 본문:

```drv
|x| {
    int doubled = x * 2;
    return doubled + 1;
}
```

### 캡처 가이드 — `[copy]` / `[ref]`

drv는 C++로 트랜스파일되므로 람다가 외부 변수를 참조할 때 **어떻게 캡처할지** 명시해야 합니다.  
명시하지 않으면 컴파일러가 수명을 분석해 자동 결정하지만, 명확한 의도 전달을 위해 캡처 가이드를 권장합니다.

```
[copy 변수, ...] |매개변수| -> 표현식    # 값 복사 캡처
[ref 변수, ...]  |매개변수| -> 표현식    # 참조 캡처 (수명 주의)
```

```drv
int factor = 10;

# 값 복사 — factor가 나중에 바뀌어도 람다 결과에 영향 없음
var mul_copy = [copy factor] |int x| -> x * factor;

# 참조 캡처 — 복사 오버헤드 없음, 단 factor 수명이 람다보다 길어야 함
var mul_ref = [ref factor] |int x| -> x * factor;

print(mul_copy(5));   # 50
print(mul_ref(5));    # 50

factor = 99;
print(mul_copy(5));   # 50  — 복사본이므로 변화 없음
print(mul_ref(5));    # 495 — 참조이므로 현재 factor 값 반영
```

| 캡처 방식 | C++ 변환 | 사용 시점 |
|----------|----------|----------|
| `[copy x]` | `[x]` | 람다가 오래 살거나, 원본 변경 무관할 때 |
| `[ref x]` | `[&x]` | 성능이 중요하고 수명이 보장될 때 |
| 미명시 | `[=]` 또는 분석 결과 | 간단한 일회성 람다 |

## 리스트 고차 함수

### .map() — 각 요소 변환

```drv
list<int> nums = [];
lst.push(nums, 1);
lst.push(nums, 2);
lst.push(nums, 3);
lst.push(nums, 4);
lst.push(nums, 5);

var doubled = nums.map(|x| -> x * 2);
# [2, 4, 6, 8, 10]

var squared = nums.map(|x| -> x * x);
# [1, 4, 9, 16, 25]
```

### .filter() — 조건 필터링

```drv
var evens = nums.filter(|x| -> x % 2 == 0);
# [2, 4]

var big = nums.filter(|x| -> x > 3);
# [4, 5]
```

### .reduce() — 누적 축약

```drv
int sum = nums.reduce(0, |acc, x| -> acc + x);
# 15

int product = nums.reduce(1, |acc, x| -> acc * x);
# 120

int max_val = nums.reduce(0, |acc, x| -> max(acc, x));
# 5
```

### .take() — 앞에서 N개

```drv
var first3 = nums.take(3);
# [1, 2, 3]
```

## 고차 함수 체이닝

고차 함수를 연결해 데이터 파이프라인을 구성합니다.

```drv
list<int> nums = [];
lst.push(nums, 1);
lst.push(nums, 2);
lst.push(nums, 3);
lst.push(nums, 4);
lst.push(nums, 5);
lst.push(nums, 6);
lst.push(nums, 7);
lst.push(nums, 8);
lst.push(nums, 9);
lst.push(nums, 10);

# 짝수만 골라 제곱 후 합산
int result = nums
    .filter(|x| -> x % 2 == 0)
    .map(|x| -> x * x)
    .reduce(0, |acc, x| -> acc + x);
# 220 = 4+16+36+64+100
print(result);
```

## 파이프 연산자 — `|>`

```
표현식 |> 함수
```

`value |> f` 는 `f(value)` 와 동일합니다.

```drv
int result = 5 |> |x| -> x * 2;    # 10
```

체이닝:

```drv
double r = 16.0
    |> |x| -> sqrt(x)          # 4.0
    |> |x| -> x * 3.14159;     # 12.566...
print(r);
```

문자열 처리 파이프라인:

```drv
String processed = "  Hello, World!  "
    |> |s| -> trim(s)
    |> |s| -> toLower(s);
print(processed);   # "hello, world!"
```

### 타입 추론 한계 — 명시적 타입 어노테이션

단일 매개변수 파이프라인은 컴파일러가 타입을 자동 추론합니다.  
**제네릭이나 고차 함수가 중간에 끼면 추론이 실패**할 수 있습니다. 이 경우 람다 매개변수에 타입을 명시합니다.

```drv
# 추론 가능: 앞 표현식이 int → x는 int로 결정
int r = 5 |> |x| -> x * 2;

# 추론 불가능 예시: list<T>.map에서 T를 확정할 수 없을 때
# 컴파일 오류 — x의 타입 불명
var result = some_list |> |x| -> x * 2;

# 해결: 람다 매개변수에 타입 명시
var result = some_list |> |int x| -> x * 2;
```

명시적 타입 람다 문법:

```
|타입 매개변수| -> 표현식
|타입 매개변수, 타입 매개변수| -> 표현식
```

```drv
# 복잡한 파이프라인 — 타입 명시
list<double> r = raw_data
    |> |list<int> l| -> l.filter(|int x| -> x > 0)
    |> |list<int> l| -> l.map(|int x| -> x as double * 1.5);
```

> 컴파일러가 타입 어노테이션 없이 추론 실패 시 **명확한 오류 메시지**를 출력합니다.  
> 단순 변환 파이프라인은 대부분 어노테이션 없이 동작합니다.

## lazy — 지연 평가

`lazy` 키워드를 붙이면 실제로 사용할 때까지 평가를 미룹니다.

```drv
lazy list large_data = expensive_computation();
# expensive_computation()은 large_data를 실제로 사용할 때만 실행됨
```

## 실용 예시

### 데이터 집계

```drv
list<double> temps = [];
lst.push(temps, 22.5);
lst.push(temps, 18.3);
lst.push(temps, 25.7);
lst.push(temps, 20.1);
lst.push(temps, 23.8);

double total = temps.reduce(0.0, |acc, x| -> acc + x);
double avg = total / lst.size(temps) as double;
print("평균:", avg);

double max_t = temps.reduce(temps[0], |acc, x| -> max(acc, x));
print("최고:", max_t);

var hot = temps.filter(|t| -> t > 23.0);
print("23도 이상:", lst.size(hot), "일");
```

### 성적 처리

```drv
list<int> scores = [];
lst.push(scores, 85);
lst.push(scores, 92);
lst.push(scores, 78);
lst.push(scores, 95);
lst.push(scores, 60);

var passed = scores.filter(|s| -> s >= 70);
int pass_sum = passed.reduce(0, |acc, x| -> acc + x);
double pass_avg = pass_sum as double / lst.size(passed) as double;
print("통과자:", lst.size(passed));
print("통과 평균:", pass_avg);
```

### 문자열 처리

```drv
list<String> raw = [];
lst.push(raw, "  Alice  ");
lst.push(raw, "BOB");
lst.push(raw, "  charlie");

var clean = raw
    .map(|s| -> trim(s))
    .map(|s| -> toUpper(s))
    .filter(|s| -> length(s) > 3);

for (name of clean) {
    print(name);
};
# ALICE, CHARLIE
```
