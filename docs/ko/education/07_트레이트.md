# 트레이트와 impl
> `trait`은 타입이 반드시 구현해야 할 메서드 집합을 정의한다. `impl`은 특정 클래스에 트레이트를 구현한다.

---

## trait — 인터페이스 정의

```
trait 이름 {
    반환타입 메서드명(매개변수);
};
```

```dri
trait Printable {
    void print_info();
};

trait Comparable {
    boolean equals(Self other);
    boolean less_than(Self other);
};

trait Serializable {
    String to_json();
};
```

---

## impl — 트레이트 구현

```
impl 트레이트명 for 클래스명 {
    반환타입 메서드명(매개변수) {
        본문
    };
};
```

```dri
class Point {
    public int x;
    public int y;
};

impl Printable for Point {
    void print_info() {
        print("Point(", this.x, ",", this.y, ")");
    };
};

impl Comparable for Point {
    boolean equals(Self other) {
        return this.x == other.x and this.y == other.y;
    };

    boolean less_than(Self other) {
        return this.x < other.x or (this.x == other.x and this.y < other.y);
    };
};
```

```dri
Point p1 = new Point();
p1.x = 3;
p1.y = 4;

Point p2 = new Point();
p2.x = 1;
p2.y = 2;

p1.print_info();               # Point( 3 , 4 )
print(p1.equals(p2));          # false
print(p2.less_than(p1));       # true
```

## 여러 트레이트 구현

하나의 클래스에 여러 트레이트를 독립적으로 구현할 수 있습니다.

```dri
trait Area {
    double area();
};

trait Perimeter {
    double perimeter();
};

trait Describe {
    void describe();
};

class Circle {
    public double radius;
};

impl Area for Circle {
    double area() {
        return 3.14159 * this.radius * this.radius;
    };
};

impl Perimeter for Circle {
    double perimeter() {
        return 2.0 * 3.14159 * this.radius;
    };
};

impl Describe for Circle {
    void describe() {
        print("Circle r =", this.radius,
              "area =", this.area(),
              "perimeter =", this.perimeter());
    };
};

Circle c = new Circle();
c.radius = 5.0;
c.describe();
```

## `Self` 키워드

트레이트 메서드 내에서 구현 타입 자기 자신을 가리킵니다.

```dri
trait Cloneable {
    Self clone();
};

class Config {
    public String host;
    public int port;
};

impl Cloneable for Config {
    Self clone() {
        Config c = new Config();
        c.host = this.host;
        c.port = this.port;
        return c;
    };
};

Config original = new Config();
original.host = "localhost";
original.port = 8080;

Config copy = original.clone();
copy.port = 9090;

print(original.port);   # 8080
print(copy.port);       # 9090
```

## 트레이트 바운드 — 제네릭 타입 제약

특정 트레이트를 구현한 타입만 인수로 받는 제네릭 함수를 선언합니다.  
C++20 Concept, Rust Trait Bound와 동일한 역할입니다.

```
반환타입 함수명<T implements 트레이트명>(T 매개변수) { 본문 }
```

```dri
trait Printable {
    void print_info();
};

trait Comparable {
    boolean less_than(Self other);
};

# T는 반드시 Printable을 구현해야 함 — 아니면 컴파일 오류
void display<T implements Printable>(T item) {
    item.print_info();
};
```

여러 트레이트 동시 요구:

```
T implements 트레이트A + 트레이트B
```

```dri
# T는 Printable과 Comparable을 모두 구현해야 함
void print_sorted<T implements Printable + Comparable>(T a, T b) {
    if (a.less_than(b)) {
        a.print_info();
        b.print_info();
    } else {
        b.print_info();
        a.print_info();
    };
};
```

반환 타입에도 적용:

```dri
T max_of<T implements Comparable>(T a, T b) {
    if (b.less_than(a)) {
        return a;
    };
    return b;
};
```

### 제약 없는 제네릭 vs 바운드 제네릭 비교

```dri
# 제약 없음 — T에 메서드 호출 불가 (identity 함수 등에만 유용)
T identity<T>(T x) {
    return x;
};

# 바운드 있음 — T의 트레이트 메서드 호출 가능
void show<T implements Printable>(T x) {
    x.print_info();   # Printable 보장되므로 OK
};
```

### 컴파일러 오류 메시지 규정

```dri
class Foo { public int x; };  # Printable 미구현

display(Foo());  # 컴파일 오류:
# error: 'Foo' does not implement trait 'Printable'
#        required by function 'display<T implements Printable>'
```

## 전체 예시 — 도형 계산 시스템

```dri
trait Shape {
    double area();
    double perimeter();
    String name();
};

class Circle {
    public double r;
};

impl Shape for Circle {
    double area() {
        return 3.14159 * this.r * this.r;
    };
    double perimeter() {
        return 2.0 * 3.14159 * this.r;
    };
    String name() {
        return "Circle";
    };
};

class Rect {
    public double w;
    public double h;
};

impl Shape for Rect {
    double area() {
        return this.w * this.h;
    };
    double perimeter() {
        return 2.0 * (this.w + this.h);
    };
    String name() {
        return "Rectangle";
    };
};

Circle c = new Circle();
c.r = 3.0;
print(c.name(), "면적:", c.area(), "둘레:", c.perimeter());

Rect r = new Rect();
r.w = 4.0;
r.h = 5.0;
print(r.name(), "면적:", r.area(), "둘레:", r.perimeter());
```
