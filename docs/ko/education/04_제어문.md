# 제어문

> 모든 제어 구조는 중괄호 `{ }` 블록을 사용하고, 문장 전체가 `;`으로 끝난다.

---

## if / else if / else문

```
if (조건) { 본문 };
if (조건) { 본문 } else { 본문 };
if (조건) { 본문 } else if (조건2) { 본문 } else { 본문 };
```

```drv
int score = 85;

if (score >= 90) {
    print("A");
} else if (score >= 80) {
    print("B");
} else if (score >= 70) {
    print("C");
} else {
    print("F");
};
# 출력: B
```

```drv
int x = 15;
if (x > 10 and x < 20) {
    print("10과 20 사이");
};

boolean logged_in = true;
boolean is_admin = false;
if (logged_in and not is_admin) {
    print("일반 사용자");
};
```

## for문

```
for (변수 in 시작..끝) { 본문 };
```

`시작..끝`은 시작 이상, 끝 **미만** (끝 포함 안 됨).

```drv
# 0, 1, 2, 3, 4
for (i in 0..5) {
    print(i);
};

# 합계 계산
int sum = 0;
for (i in 1..101) {
    sum += i;
};
print("1~100 합:", sum);   # 5050
```

### for — 컬렉션 반복

```
for (변수 of 컬렉션) { 본문 };
```

```drv
int[] numbers = [10, 20, 30, 40, 50];
for (n of numbers) {
    print(n);
};
```

### for — C 스타일 명시 반복

```
for (타입 변수 = 초기값; 조건; 변수++) { 본문 };
```

```drv
for (int i = 0; i < 10; i++) {
    print(i);
};

for (int i = 10; i > 0; i--) {
    print(i);
};

for (int i = 0; i < 100; i += 5) {
    print(i);
};
```

## while문

```
while (조건) { 본문 };
```

```drv
int count = 0;
while (count < 5) {
    print(count);
    count += 1;
};

# 무한 루프
int n = 1;
while (true) {
    if (n > 10) {
        break;
    };
    print(n);
    n += 1;
};
```

## break / continue / pass

```drv
# break — 루프 탈출
for (i in 0..10) {
    if (i == 5) {
        break;
    };
    print(i);
};
# 출력: 0 1 2 3 4

# continue — 현재 반복 건너뜀
for (i in 0..10) {
    if (i % 2 == 0) {
        continue;
    };
    print(i);
};
# 출력: 1 3 5 7 9

# pass — 아무것도 하지 않음 (빈 블록 대용)
for (i in 0..5) {
    pass;
};
```

## switch

```
switch (표현식) {
    case (값) -> { 본문 };
    case (값2) -> { 본문 };
};
```

```drv
int day = 3;
switch (day) {
    case (1) -> { print("월요일"); };
    case (2) -> { print("화요일"); };
    case (3) -> { print("수요일"); };
    case (4) -> { print("목요일"); };
    case (5) -> { print("금요일"); };
    case default -> { print("휴일"); };
};
# 출력: 수요일
```

### 문자열 switch:

```drv
String cmd = "quit";
switch (cmd) {
    case ("start") -> { print("시작"); };
    case ("stop")  -> { print("정지"); };
    case ("quit")  -> { print("종료"); };
};
```

## match — 패턴 매칭

`Option<T>`, `Result<T>` 값을 분해할 때 사용합니다.

```
match 표현식 {
    패턴 => { 본문 };
    패턴 => { 본문 };
};
```

패턴 종류: `Some(변수)`, `None`, `Ok(변수)`, `Err(변수)`, `_` (와일드카드), 리터럴

```drv
Option<int> maybe = Some(42);
match maybe {
    Some(v) => { print("값:", v); };
    None    => { print("없음"); };
};
# 출력: 값: 42

Result<int> res = Ok(100);
match res {
    Ok(v)  => { print("성공:", v); };
    Err(e) => { print("실패:", e); };
};
```

### 와일드카드 패턴:

```drv
match maybe {
    Some(v) => { print("있음:", v); };
    _       => { print("없거나 다른 패턴"); };
};
```

### 리터럴 패턴:

```drv
int code = 404;
match code {
    200 => { print("OK"); };
    404 => { print("Not Found"); };
    500 => { print("Server Error"); };
    _   => { print("기타:", code); };
};
```

## parallel for — 병렬 반복

`for` 앞에 접두사를 붙여 실행 방식을 변경합니다.

```
parallel for (변수 of 컬렉션) { 본문 };
parallel for (변수 in 시작..끝) { 본문 };
```

Work-stealing 스레드 풀 기반으로 각 반복을 병렬 실행합니다.

```drv
int[] data = [1, 2, 3, 4, 5, 6, 7, 8];

parallel for (item of data) {
    # 각 반복이 다른 스레드에서 실행될 수 있음
    print("처리:", item);
};
```

`simd for`와 조합하면 병렬 + 벡터화를 동시에 적용할 수 있습니다:

```drv
parallel simd for (i in 0..N) {
    result[i] = a[i] * b[i] + c[i];
};
```

## simd for — SIMD 벡터화 반복

```
simd for (변수 in 시작..끝) { 본문 };
```

```drv
int N = 1024;
double[] a = new double[N];
double[] b = new double[N];
double[] c = new double[N];

simd for (i in 0..N) {
    c[i] = a[i] + b[i];
};
```

### where / otherwise — SIMD 마스크 조건

`simd for` 내부에 일반 `if/else`를 쓰면 분기 예측이 깨지고 SIMD 벡터화가 풀립니다.  
`where`/`otherwise`는 **AVX-512 Mask Register** 또는 `vblendv` 등 하드웨어 마스크 명령으로 변환됩니다.

```
simd for (변수 in 범위) {
    where (마스크 조건) {
        본문_참
    } otherwise {
        본문_거짓
    };
};
```

```drv
# 조건 분기 없이 SIMD 레인 전체를 벡터 마스크로 처리
simd for (i in 0..N) {
    where (x[i] > 0.0) {
        z[i] = sqrt(x[i]);
    } otherwise {
        z[i] = 0.0;
    };
};
# C++: z[i] = _mm512_mask_sqrt_pd(zero, mask, x[i]) 형태로 변환
```

여러 마스크 중첩:

```drv
simd for (i in 0..N) {
    where (a[i] > 0.0 and b[i] < 1.0) {
        c[i] = a[i] * b[i];
    } otherwise {
        c[i] = 0.0;
    };
};
```

> `where`/`otherwise`는 `simd for` 내부에서만 유효합니다.  
> 일반 `for` 내부에서는 일반 `if/else`를 사용하세요.

## exists — 변수 존재 확인

```
exists (변수명) { 본문 } else { 본문 };
```

```drv
String data;
exists (data) {
    print("data 존재:", data);
} else {
    print("data 없음");
};
```
