# 연산자

---

## 산술 연산자

```drv
int a = 10;
int b = 3;

print(a + b);    # 13
print(a - b);    # 7
print(a * b);    # 30
print(a / b);    # 3.33333...
print(a // b);   # 3 (정수 나눗셈)
print(a % b);    # 1  (나머지)
print(a ** b);   # 1000 (거듭제곱)

double x = 10.0;
double y = 3.0;
print(x / y);
```
> `//` 연산자로 변수에 대입할 때는 float 타입으로 받아야 합니다. (타입 오류 방지)

## 복합 대입 연산자

```drv
int n = 10;
n += 5;    # n = 15
n -= 3;    # n = 12
n *= 2;    # n = 24
n /= 4;    # n = 6
n %= 4;    # n = 2
```
> `++`, `--`도 사용할 수 있습니다.

## 비교 연산자

```drv
int a = 5;
int b = 10;

print(a == b);   # false
print(a != b);   # true
print(a < b);    # true
print(a > b);    # false
print(a <= b);   # true
print(a >= b);   # false
```

## 논리 연산자

`and`, `or`, `not`이 정규 키워드입니다. `&&`, `||`, `!`는 하위 호환 별칭이므로 새 코드에서는 사용을 지양합니다.

```drv
boolean t = true;
boolean f = false;

print(t and f);    # false
print(t or f);     # true
print(not t);      # false

int x = 15;
print(x > 10 and x < 20);   # true
print(x < 5 or x > 10);     # true
```

> `&&`, `||`, `!`는 파서가 인식하지만 **비권장(deprecated)** 별칭입니다.  
> 비트 연산자 `&`, `|`와 시각적으로 구분하기 위해 키워드 형태를 정규 문법으로 채택했습니다.

---

## 비트 연산자

```drv
int a = 0b1100;   # 12
int b = 0b1010;   # 10

print(a & b);    # 0b1000 = 8  (AND)
print(a | b);    # 0b1110 = 14 (OR)
print(a ^ b);    # 0b0110 = 6  (XOR)
print(~a);       # 비트 반전 (NOT)
print(a << 2);   # 0b110000 = 48 (왼쪽 시프트)
print(a >> 1);   # 0b0110 = 6   (오른쪽 시프트)
```

| 연산자 | 이름 | 설명 |
|--------|------|------|
| `&` | AND | 두 비트 모두 1이면 1 |
| `\|` | OR | 둘 중 하나라도 1이면 1 |
| `^` | XOR | 두 비트가 다르면 1 |
| `~` | NOT | 비트 반전 |
| `<<` | 왼쪽 시프트 | 2^n 곱셈 효과 |
| `>>` | 오른쪽 시프트 | 2^n 나눗셈 효과 |

## `in` 연산자 (멤버십 테스트)

```drv
int[] arr = [1, 2, 3, 4, 5];
print(3 in arr);    # true
print(9 in arr);    # false
```

---

## `is` 연산자

```drv
int x = 5;
print(x is 5);      # true (== 와 동일)
print(x is not 3);  # true (!= 와 동일)
```

## 타입 캐스팅 — `as`

```drv
int n = 42;
double d = n as double;     # 42.0
int back = d as int;        # 42 (소수점 버림)

long big = 9999999999;
int small = big as int;     # 오버플로 주의
```

## 리터럴 종류

```drv
# 정수
int dec = 255;
int hex = 0xFF;       # 16진수 (255)
int bin = 0b11001100; # 2진수

# 부동소수점
double pi = 3.14159;
float small = 1.5;

# 문자열 (큰따옴표)
String s = "hello";

# 문자 (작은따옴표)
char c = 'A';

# 불린
boolean yes = true;
boolean no = false;

# null (참조형)
Ref<String> ref = null;
```

## 연산자 우선순위

높은 순서대로 나열합니다.

| 순위 | 연산자 |
|------|--------|
| 1 | `**` (거듭제곱) |
| 2 | `~` (비트 NOT), 단항 `-` |
| 3 | `*`, `/`, `//`, `%` |
| 4 | `+`, `-` |
| 5 | `<<`, `>>` |
| 6 | `&` |
| 7 | `^` |
| 8 | `\|` |
| 9 | `==`, `!=`, `<`, `>`, `<=`, `>=`, `in`, `is` |
| 10 | `not` |
| 11 | `and` |
| 12 | `or` |
