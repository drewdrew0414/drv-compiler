## 함수 기본 선언
```drv
수식자 반환타입 함수명(타입 매개변수, ...) {
    본문
};
```
### 수식자
| 수식자 | 설명                                                               |
| --- |------------------------------------------------------------------|
| public | 모든 파일에서 쓸 수 있게 합니다.                                               |
| private | 해당 파일에서만 쓸 수 있게 합니다.                                              |
| static | 객체(인스턴스)를 생성하지 않고도 클래스명.메서드명()으로 바로 호출할 수 있게 만듭니다                                                                 |
| abstract | 추상 메서드를 만들 때 씁니다. 내용물{} 없이 선언만 하고, 상속받은 자식 클래스가 반드시 구현하도록 강제합니다. | 
| final | 오버라이딩(재정의)을 금지합니다. 자식 클래스에서 이 메서드를 바꿀 수 없게 예약해 두는 것입니다. |
| synchronized | 멀티쓰레드 환경에서 동기화를 보장합니다. 한 쓰레드가 이 메서드를 쓰는 동안 다른 쓰레드는 대기하게 만듭니다. |

### 반환값
```drv
# 단일 반환
int max_of(int a, int b) {
    if (a > b) {
        return a;
    };
    return b;
};

# 조기 반환
int safe_divide(int a, int b) {
    if (b == 0) {
        return 0;
    };
    return a / b;
};

print(max_of(10, 25));       # 25
print(safe_divide(10, 0));   # 0
print(safe_divide(10, 3));   # 3
```

## 매개변수 기본값

```drv
int power(int base, int exp = 2) {
    int result = 1;
    for (i in 0..exp) {
        result *= base;
    };
    return result;
};

print(power(3));      # 9 (기본값 exp=2)
print(power(3, 3));   # 27
```

## 재귀 함수

```drv
int factorial(int n) {
    if (n <= 1) {
        return 1;
    };
    return n * factorial(n - 1);
};

int fib(int n) {
    if (n <= 1) {
        return n;
    };
    return fib(n - 1) + fib(n - 2);
};

print(factorial(5));   # 120
print(fib(10));        # 55
```

## 클래스

### 기본 클래스

```
class 이름 {
    public 타입 필드명;
    public 반환타입 메서드명(매개변수) {
        본문
    };
};
```

> 클래스 선언 전체는 `;`으로 끝납니다. 필드·메서드도 각각 `;`으로 끝납니다.

```drv
class Point {
    public int x;
    public int y;

    public void move(int dx, int dy) {
        this.x += dx;
        this.y += dy;
    };

    public String to_string() {
        return "(" + str.from_int(this.x) + ", " + str.from_int(this.y) + ")";
    };
};
```

### 인스턴스 생성

```drv
Point p = new Point();
p.x = 3;
p.y = 7;

p.move(2, -1);
print(p.to_string());    # (5, 6)
```

## `this` 키워드

클래스 메서드 내에서 현재 인스턴스의 필드에 접근할 때 사용합니다.

```drv
class Counter {
    public int value;

    public void increment() {
        this.value += 1;
    };

    public void add(int n) {
        this.value += n;
    };

    public int get() {
        return this.value;
    };
};

Counter c = new Counter();
c.value = 0;
c.increment();
c.increment();
c.add(5);
print(c.get());   # 7
```

## 상속 — `extends`

```drv
class Animal {
    public String name;
    public int age;

    public void describe() {
        print("이름:", this.name, "나이:", this.age);
    };

    public void speak() {
        print("...");
    };
};

class Dog extends Animal {
    public String breed;

    override void speak() {
        print("멍멍!");
    };

    public void fetch() {
        print(this.name, "이(가) 공을 가져옵니다.");
    };
};

class Cat extends Animal {
    override void speak() {
        print("야옹~");
    };
};
```

```drv
Dog d = new Dog();
d.name = "바둑이";
d.age = 3;
d.breed = "진돗개";
d.describe();    # 이름: 바둑이 나이: 3
d.speak();       # 멍멍!
d.fetch();       # 바둑이 이(가) 공을 가져옵니다.

Cat c = new Cat();
c.name = "나비";
c.age = 2;
c.speak();       # 야옹~
```

## 추상 클래스 — `abstract class`

인스턴스화할 수 없고, 자식 클래스가 반드시 구현해야 합니다.  
추상 메서드는 `@abstract` 어노테이션을 사용합니다.

```drv
abstract class Shape {
    public String color;

    @abstract double area();
    @abstract double perimeter();

    public void describe() {
        print("도형:", this.color,
              "면적:", this.area(),
              "둘레:", this.perimeter());
    };
};

class Circle extends Shape {
    public double radius;

    override double area() {
        return 3.14159 * this.radius * this.radius;
    };

    override double perimeter() {
        return 2.0 * 3.14159 * this.radius;
    };
};

class Rectangle extends Shape {
    public double width;
    public double height;

    override double area() {
        return this.width * this.height;
    };

    override double perimeter() {
        return 2.0 * (this.width + this.height);
    };
};
```

```drv
Circle circ = new Circle();
circ.color = "빨강";
circ.radius = 5.0;
circ.describe();

Rectangle rect = new Rectangle();
rect.color = "파랑";
rect.width = 4.0;
rect.height = 6.0;
rect.describe();
```

## `@packed` — 패딩 없는 클래스

필드 사이 패딩을 제거해 메모리를 절약합니다.

```drv
@packed
class Pixel {
    public char r;
    public char g;
    public char b;
    public char a;    # 4바이트 밀착, 패딩 없음
};
```

> **제약**: `@packed` 클래스는 **상속에 참여할 수 없습니다.**  
> 상속된 클래스는 vtable 포인터를 포함하는데, `@packed`는 그 포인터의 정렬(Alignment)을 깨뜨려  
> CPU 레벨 Misaligned Access 또는 크래시를 유발합니다.  
> 컴파일러는 아래 두 경우 모두 **컴파일 오류**를 발생시킵니다.

```drv
# 오류: @packed 클래스는 다른 클래스를 상속할 수 없음
@packed
class PackedCircle extends Shape { ... };   # 컴파일 에러

# 오류: @packed 클래스를 상속할 수 없음
class Child extends Pixel { ... };          # 컴파일 에러
```

`@packed`는 **최종 리프(leaf) 데이터 구조** — 즉 상속이 없는 순수 데이터 클래스에만 사용하세요.

## 필드 체인 접근

```drv
class Engine {
    public int horsepower;
};

class Car {
    public String name;
    public Engine engine;
};

Car car = new Car();
car.name = "Turbo";
car.engine = new Engine();
car.engine.horsepower = 300;

print(car.engine.horsepower);    # 300
car.engine.horsepower = 350;     # 체인 대입
```
