## drv-pandas — 데이터프레임 및 파이프라인 연산

### 백엔드

Apache Arrow 또는 Rust Polars 코어를 `extern "FFI"`로 래핑합니다.  
열 기반 메모리 레이아웃(`@layout_soa`)과 제로 카피 슬라이싱으로 대용량 데이터를 처리합니다.

```drv
use drv_pandas;

# CSV 로드
DataFrame df = pandas.read_csv("sales.csv");

# 쿼리 및 변환
DataFrame result = df
    .query("revenue > 1000")
    .assign("profit", |row| -> row["revenue"] - row["cost"])
    .groupby("region")
    .mean("profit");

print(result);
```

### 핵심 메서드

| 메서드 | 설명 |
|--------|------|
| `pandas.read_csv(path)` | CSV 파일 → DataFrame |
| `pandas.read_json(path)` | JSON 파일 → DataFrame |
| `df.query(expr)` | 조건 필터링 |
| `df.assign(col, lambda)` | 파생 열 추가 |
| `df.groupby(col)` | 그룹 집계 시작 |
| `df.mean(col)` | 열 평균 |
| `df.sum(col)` | 열 합계 |
| `df.get_column(name)` | 단일 열 추출 → Series |
| `df.head(n)` | 상위 n행 |
| `df.shape()` | (행 수, 열 수) |
| `df.to_tensor()` | 수치 열 → `tensor` 변환 |

### 메서드 체이닝

```drv
use drv_pandas;

DataFrame df = pandas.read_csv("students.csv");

DataFrame top = df
    .query("score >= 90")
    .assign("grade", |row| -> "A")
    .groupby("class")
    .mean("score");

for (row of top.iter_rows()) {
    print(row["class"], "평균:", row["score"]);
};
```

## drv-toulmin — 논증 기반 룰 엔진

> 참조: [park-jun-woo/toulmin](https://github.com/park-jun-woo/toulmin)

### 개요

툴민(Toulmin) 논증 모델에 기반한 규칙 평가 엔진입니다.  
`@warrant`, `@rebuttal`, `@defeats` 어노테이션으로 정의된 규칙 그래프를 런타임에 로드하고,  
신뢰도 점수(Acceptance Score)를 계산해 최종 판단을 내립니다.

### 신뢰도 계산 알고리즘

$$Acc(a) = \frac{w(a)}{1 + \sum_{b \in attackers(a)} Acc(b)}$$

- `w(a)` — 규칙 a의 기본 가중치
- `attackers(a)` — a를 공격(rebuttal/defeats)하는 규칙 집합
- 반복 계산으로 수렴 (기본 10회 반복)

### 사용법

```drv
use drv_toulmin;

# @warrant / @rebuttal / @defeats 어노테이션으로 규칙 정의
@warrant("is_approved")
boolean income_check(double income) {
    return income >= 3000000.0;
};

@warrant("is_approved")
boolean credit_check(int credit_score) {
    return credit_score >= 700;
};

@rebuttal("is_approved")
boolean has_debt(double debt_ratio) {
    return debt_ratio > 0.5;
};

@defeats("has_debt")
boolean collateral_exists(boolean has_collateral) {
    return has_collateral;
};

# 룰 엔진 초기화 — 바이너리 메타데이터에서 규칙 그래프 자동 로드
RuleEngine engine = toulmin.load_rules();

# 입력 데이터
Map<String, double> facts = {};
map.set(facts, "income", 3500000.0);
map.set(facts, "credit_score", 720.0);
map.set(facts, "debt_ratio", 0.6);
map.set(facts, "has_collateral", 1.0);

# 평가
ToulminResult result = engine.evaluate("is_approved", facts);

print("승인 여부:", result.accepted);       # true
print("신뢰도 점수:", result.score);         # 0.72
print("활성 규칙:", result.active_rules);    # ["income_check", "credit_check", "collateral_exists"]
```

### 감사 추적 (Audit Trail)

`@trace` 어노테이션과 결합하면 최종 판단의 근거 트리를 생성합니다.

```drv
use drv_toulmin;

# @trace가 붙은 함수들의 분기 이력을 논증 트리와 결합
ToulminResult result = engine.evaluate("is_approved", facts);
AuditTree audit = toulmin.build_audit(result, sys.get_branch_trace());

print(audit.to_string());
# 출력:
# is_approved [ACCEPTED, score=0.72]
# ├── income_check     [WARRANT, passed]  → income=3500000 > 3000000
# ├── credit_check     [WARRANT, passed]  → credit=720 > 700
# ├── has_debt         [REBUTTAL, fired]  → debt_ratio=0.6 > 0.5
# │   └── collateral_exists [DEFEATS has_debt, fired] → has_collateral=true
```

### 주요 API

| 함수 | 설명 |
|------|------|
| `toulmin.load_rules()` | 바이너리 메타데이터에서 규칙 그래프 로드 |
| `engine.evaluate(claim, facts)` | 주장 평가 → `ToulminResult` |
| `toulmin.build_audit(result, trace)` | 근거 트리 생성 → `AuditTree` |
| `engine.set_weight(rule, weight)` | 규칙 가중치 조정 |
| `engine.export_graph(path)` | 규칙 그래프를 JSON으로 저장 |

## drv-data — 멀티 포맷 입출력 및 변환

데이터프레임, 룰 엔진 결과, 텐서를 다양한 포맷으로 저장·불러옵니다.

### 텍스트 직렬화

```drv
use drv_data;

DataFrame df = pandas.read_csv("input.csv");

# CSV 저장
data.write_csv(df, "output.csv");

# JSON 저장
data.write_json(df, "output.json");

# XML 저장
data.write_xml(df, "output.xml");

# 툴민 결과 JSON 저장
ToulminResult result = engine.evaluate("is_approved", facts);
data.write_json(result, "audit.json");
```

### 이미지 직렬화

```drv
use drv_data;
use drv_plot;

DataFrame df = pandas.read_csv("sales.csv");

# 데이터프레임 테이블을 PNG 이미지로 내보내기
data.export_table_png(df, "table.png");

# 논증 그래프를 PNG 이미지로 내보내기
AuditTree audit = toulmin.build_audit(result, sys.get_branch_trace());
data.export_graph_png(audit, "argument_graph.png");
```

### 주요 API

| 함수 | 설명 |
|------|------|
| `data.write_csv(obj, path)` | CSV 저장 |
| `data.write_json(obj, path)` | JSON 저장 |
| `data.write_xml(obj, path)` | XML 저장 |
| `data.read_json(path)` | JSON → 타입 추론 로드 |
| `data.export_table_png(df, path)` | DataFrame 테이블 → PNG |
| `data.export_graph_png(tree, path)` | 논증 트리 → PNG |

## drv-plot — 고성능 데이터 시각화

### 차트 엔진

```drv
use drv_plot;

DataFrame df = pandas.read_csv("sales.csv");

# 선형 도표
plot.line(df, x="month", y="revenue", title="월별 매출");

# 막대 도표
plot.bar(df, x="region", y="profit", title="지역별 이익");

# 산점도
plot.scatter(df, x="cost", y="revenue", color="region");

# 히트맵 (tensor 데이터)
tensor<10, 10, double> matrix = ...;
plot.heatmap(matrix, title="상관 행렬");
```

### 대시보드 뷰어

`plot.show()`는 GUI 창 또는 웹 기반 인터페이스로 차트를 표시합니다.

```drv
use drv_plot;

DataFrame df = pandas.read_csv("monthly.csv");

# 여러 차트를 대시보드로 구성
Dashboard dash = plot.dashboard("월간 리포트");

dash.add(plot.line(df, x="month", y="revenue"));
dash.add(plot.bar(df, x="region", y="profit"));
dash.add(plot.scatter(df, x="cost", y="revenue"));

# GUI 창으로 표시
dash.show();

# HTML 파일로 내보내기
dash.export_html("dashboard.html");

# PNG 이미지로 내보내기
dash.export_png("dashboard.png");
```

### 주요 API

| 함수 | 설명 |
|------|------|
| `plot.line(df, x, y)` | 선형 도표 |
| `plot.bar(df, x, y)` | 막대 도표 |
| `plot.scatter(df, x, y)` | 산점도 |
| `plot.heatmap(tensor)` | 히트맵 |
| `plot.dashboard(title)` | 대시보드 생성 |
| `dash.add(chart)` | 차트 추가 |
| `dash.show()` | GUI 창 또는 웹 뷰어 표시 |
| `dash.export_html(path)` | HTML 내보내기 |
| `dash.export_png(path)` | PNG 내보내기 |

## 전체 파이프라인 예시

```drv
use drv_pandas;
use drv_toulmin;
use drv_data;
use drv_plot;

# 1. 데이터 로드
DataFrame df = pandas.read_csv("loan_applications.csv");

# 2. 전처리
DataFrame processed = df
    .query("age >= 18")
    .assign("debt_ratio", |row| -> row["debt"] / row["income"]);

# 3. 룰 엔진 평가
RuleEngine engine = toulmin.load_rules();
list<ToulminResult> results = [];

for (row of processed.iter_rows()) {
    Map<String, double> facts = row.to_map();
    ToulminResult r = engine.evaluate("is_approved", facts);
    lst.push(results, r);
};

# 4. 결과 시각화
DataFrame scored = processed.assign("score", |i| -> results[i].score);
plot.bar(scored, x="applicant_id", y="score", title="대출 승인 점수").show();

# 5. 감사 추적 저장
AuditTree audit = toulmin.build_audit(results[0], sys.get_branch_trace());
data.export_graph_png(audit, "audit_trail.png");
data.write_json(audit, "audit_trail.json");
```
