# 확장 함수 레퍼런스

## math.* — 확장 수학 함수

| 함수 | 인수 | 반환 | 설명 |
|------|------|------|------|
| `math.pow(x, n)` | 2 | double | x의 n제곱 |
| `math.log(x)` | 1 | double | 자연로그 ln(x) |
| `math.log2(x)` | 1 | double | 로그 밑 2 |
| `math.log10(x)` | 1 | double | 로그 밑 10 |
| `math.exp(x)` | 1 | double | e^x |
| `math.asin(x)` | 1 | double | 역사인 (라디안) |
| `math.acos(x)` | 1 | double | 역코사인 (라디안) |
| `math.atan(x)` | 1 | double | 역탄젠트 (라디안) |
| `math.atan2(y, x)` | 2 | double | 역탄젠트 (4사분면 구분) |
| `math.hypot(x, y)` | 2 | double | sqrt(x²+y²) |
| `math.clamp(x, lo, hi)` | 3 | double | lo ≤ x ≤ hi 범위 제한 |
| `math.lerp(a, b, t)` | 3 | double | 선형 보간: a+(b-a)\*t |
| `math.pi()` | 0 | double | π ≈ 3.14159... |
| `math.e()` | 0 | double | e ≈ 2.71828... |
| `math.sign(x)` | 1 | int | 부호: -1 / 0 / 1 |
| `math.degrees(x)` | 1 | double | 라디안 → 도 |
| `math.radians(x)` | 1 | double | 도 → 라디안 |
| `math.gcd(a, b)` | 2 | long | 최대공약수 |
| `math.lcm(a, b)` | 2 | long | 최소공배수 |
| `math.factorial(n)` | 1 | long | n! |
| `math.is_nan(x)` | 1 | boolean | NaN 여부 |
| `math.is_inf(x)` | 1 | boolean | Infinity 여부 |
| `math.floor_div(a, b)` | 2 | long | 바닥 나눗셈 floor(a/b) |
| `math.sinh(x)` | 1 | double | 쌍곡사인 |
| `math.cosh(x)` | 1 | double | 쌍곡코사인 |
| `math.tanh(x)` | 1 | double | 쌍곡탄젠트 |

```dri
double area = math.pi() * r * r;
double dist = math.hypot(3.0, 4.0);       # 5.0
double angle = math.atan2(1.0, 1.0);      # π/4
double clamped = math.clamp(x, 0.0, 1.0);
double lerped = math.lerp(0.0, 100.0, 0.5);  # 50.0
double powered = math.pow(2.0, 10.0);     # 1024.0
double logged = math.log(math.e());       # 1.0
int sign = math.sign(-5.0);              # -1

double deg = math.degrees(math.pi());     # 180.0
long g = math.gcd(12, 18);               # 6
long f = math.factorial(5);              # 120
long fd = math.floor_div(-7, 2);         # -4
double sh = math.sinh(1.0);              # ~1.175
```

## str.* — 확장 문자열 함수

| 함수 | 인수 | 반환 | 설명 |
|------|------|------|------|
| `str.join(lst, sep)` | 2 | String | 리스트의 문자열을 sep으로 결합 |
| `str.index_of(s, sub)` | 2 | long | 부분 문자열 위치 (-1: 없음) |
| `str.pad_left(s, n)` | 2 | String | 왼쪽에 공백을 채워 n자 맞춤 |
| `str.pad_right(s, n)` | 2 | String | 오른쪽에 공백을 채워 n자 맞춤 |
| `str.is_empty(s)` | 1 | boolean | 빈 문자열 여부 |
| `str.count(s, sub)` | 2 | long | sub의 등장 횟수 |
| `str.reverse(s)` | 1 | String | 문자열 역순 |
| `str.to_float(s)` | 1 | double | 문자열 → double 파싱 |
| `str.from_float(x)` | 1 | String | double → 문자열 변환 |
| `str.from_bool(b)` | 1 | String | `"true"` 또는 `"false"` |
| `str.to_bool(s)` | 1 | boolean | `"true"`/`"1"` → true |
| `str.from_long(n)` | 1 | String | long → 문자열 변환 |
| `str.left(s, n)` | 2 | String | 앞 n글자 |
| `str.right(s, n)` | 2 | String | 뒤 n글자 |
| `str.char_code(c)` | 1 | long | ASCII 코드 |
| `str.from_char_code(n)` | 1 | String | ASCII 코드 → 문자 |
| `str.upper_first(s)` | 1 | String | 첫 글자만 대문자 |
| `str.trim_left(s)` | 1 | String | 왼쪽 공백만 제거 |
| `str.trim_right(s)` | 1 | String | 오른쪽 공백만 제거 |
| `str.remove(s, sub)` | 2 | String | sub의 모든 등장 제거 |
| `str.truncate(s, n)` | 2 | String | 앞 n글자만 남김 |

```dri
list<String> words = [];
lst.push(words, "hello");
lst.push(words, "world");
lst.push(words, "dri");

String joined = str.join(words, ", ");   # "hello, world, dri"
print(joined);

long idx = str.index_of("hello world", "world");   # 6

print(str.pad_left("42", 6));     # "    42"
print(str.pad_right("hi", 5));    # "hi   "
print(str.is_empty(""));          # true
print(str.count("banana", "an")); # 2
print(str.reverse("hello"));      # "olleh"

double pi = str.to_float("3.14159");
print(str.from_float(pi));        # "3.14159"
print(str.from_bool(true));       # "true"
print(str.from_long(9999999999)); # "9999999999"

String s = "Hello, World!";
print(str.left(s, 5));            # "Hello"
print(str.right(s, 6));          # "World!"
print(str.upper_first("hello dri")); # "Hello dri"
print(str.trim_left("   hello"));    # "hello"
print(str.remove("banana", "an"));   # "ba"
print(str.truncate("Hello, World!", 5)); # "Hello"
```

## lst.* — 확장 리스트 함수

### 변형 함수 (void)

| 함수 | 인수 | 설명 |
|------|------|------|
| `lst.insert(l, i, val)` | 3 | i번 위치에 val 삽입 |
| `lst.remove(l, i)` | 2 | i번 요소 삭제 |
| `lst.clear(l)` | 1 | 모든 요소 삭제 |
| `lst.extend(l, other)` | 2 | other를 l 끝에 이어 붙이기 |

```dri
list<String> fruits = [];
lst.push(fruits, "apple");
lst.push(fruits, "cherry");

lst.insert(fruits, 1, "banana");   # apple, banana, cherry
lst.remove(fruits, 0);             # banana, cherry

list<String> more = [];
lst.push(more, "date");
lst.push(more, "elderberry");
lst.extend(fruits, more);          # banana, cherry, date, elderberry

lst.clear(fruits);
print(lst.size(fruits));           # 0
```

### 조회 함수 (값 반환)

| 함수 | 인수 | 반환 | 설명 |
|------|------|------|------|
| `lst.contains(l, val)` | 2 | boolean | val 포함 여부 |
| `lst.index_of(l, val)` | 2 | long | val의 인덱스 (-1: 없음) |
| `lst.first(l)` | 1 | value | 첫 번째 요소 |
| `lst.last(l)` | 1 | value | 마지막 요소 |
| `lst.min(l)` | 1 | String | 최솟값 원소 (사전 순) |
| `lst.max(l)` | 1 | String | 최댓값 원소 (사전 순) |
| `lst.count(l, val)` | 2 | long | val의 등장 횟수 |
| `lst.is_empty(l)` | 1 | boolean | 비어 있는지 여부 |

```dri
list<String> nums = [];
lst.push(nums, "10");
lst.push(nums, "20");
lst.push(nums, "30");

print(lst.contains(nums, "20"));    # true
print(lst.index_of(nums, "20"));   # 1
print(lst.first(nums));            # "10"
print(lst.last(nums));             # "30"
print(lst.min(nums));              # "10"
print(lst.max(nums));              # "30"
print(lst.is_empty(nums));         # false
```

### 복사 반환 함수

| 함수 | 인수 | 반환 | 설명 |
|------|------|------|------|
| `lst.reverse(l)` | 1 | list | 역순 복사본 |
| `lst.sort(l)` | 1 | list | 정렬된 복사본 |
| `lst.join(l, sep)` | 2 | String | sep으로 결합한 문자열 |
| `lst.slice(l, start, end)` | 3 | list | `[start..end)` 슬라이스 |
| `lst.unique(l)` | 1 | list | 중복 제거 (순서 유지) |
| `lst.fill(n, val)` | 2 | list | n개 val로 채운 리스트 |
| `lst.copy(l)` | 1 | list | 얕은 복사본 |

```dri
list<String> data = [];
lst.push(data, "banana");
lst.push(data, "apple");
lst.push(data, "cherry");
lst.push(data, "apple");

var sorted = lst.sort(data);
print(lst.get(sorted, 0));        # "apple"

var rev = lst.reverse(data);
print(lst.get(rev, 0));           # "apple"  (역순의 첫 원소)

String sentence = lst.join(data, " + ");
print(sentence);                   # "banana + apple + cherry + apple"

var part = lst.slice(data, 1, 3);
print(lst.size(part));            # 2

var unique_fruits = lst.unique(data);
print(lst.size(unique_fruits));   # 3

var zeros = lst.fill(5, "0");
print(lst.size(zeros));           # 5
```

## map.* — 확장 맵 함수

| 함수 | 인수 | 반환 | 설명 |
|------|------|------|------|
| `map.values(m)` | 1 | list | 모든 값의 리스트 |
| `map.clear(m)` | 1 | void | 모든 항목 삭제 |
| `map.contains_value(m, val)` | 2 | boolean | 값 포함 여부 |
| `map.merge(m1, m2)` | 2 | Map | m2로 m1을 덮어씌운 복사본 |
| `map.copy(m)` | 1 | Map | 얕은 복사 |

```dri
Map<String, int> a = {};
map.set(a, "x", 1);
map.set(a, "y", 2);

Map<String, int> b = {};
map.set(b, "y", 99);
map.set(b, "z", 3);

print(map.contains_value(a, 1));    # true

var merged = map.merge(a, b);
# merged = {x:1, y:99, z:3}
print(map.get(merged, "y"));        # 99

var c = map.copy(a);
map.set(c, "w", 5);
print(map.has(a, "w"));             # false (원본 영향 없음)

map.clear(a);
print(map.size(a));                 # 0
```

## io.* — 파일시스템 확장 함수

| 함수 | 인수 | 반환 | 설명 |
|------|------|------|------|
| `io.exists(path)` | 1 | boolean | 파일/디렉토리 존재 여부 |
| `io.delete_file(path)` | 1 | boolean | 파일 삭제 |
| `io.file_size(path)` | 1 | long | 파일 크기 (바이트, -1: 오류) |
| `io.list_dir(path)` | 1 | list\<String\> | 디렉토리 목록 |
| `io.make_dir(path)` | 1 | boolean | 디렉토리 생성 (중간 경로 포함) |

```dri
if (io.exists("data.txt")) {
    long size = io.file_size("data.txt");
    print("크기:", size, "바이트");
} else {
    print("파일 없음");
};

io.write_file("temp.txt", "임시 데이터");
io.delete_file("temp.txt");

io.make_dir("output/logs");

list<String> files = io.list_dir(".");
for (f of files) {
    print(f);
};
```

```dri
String home = sys.env("HOME");

long t0 = sys.time();

int result = compute();
```

## 글로벌 유틸리티 함수

네임스페이스 없이 직접 호출할 수 있는 함수입니다.  
`sort`, `reverse`, `unique` 등 리스트 조작은 `lst.*` 네임스페이스를 사용합니다.

| 함수 | 인수 | 반환 | 설명 |
|------|------|------|------|
| `range(n)` | 1 | list | `["0","1",...,"n-1"]` 문자열 리스트 |
| `range(start, end)` | 2 | list | `[start..end)` 문자열 리스트 |
| `assert_that(cond, msg)` | 2 | void | 조건 실패 시 예외 throw |

```dri
# range — 정수 범위 리스트 생성
var r = range(5);
for (s of r) {
    print(s);    # "0", "1", "2", "3", "4"
};

var r2 = range(3, 8);
for (s of r2) {
    print(s);    # "3", "4", "5", "6", "7"
};

# assert_that — 사전 조건 검사
int divide(int a, int b) {
    assert_that(b != 0, "나눗셈의 분모는 0이 될 수 없습니다");
    return a / b;
};

try {
    int r = divide(10, 0);
} catch (err) {
    print("예외:", err);   # Assertion failed: 나눗셈의 분모는 0이 될 수 없습니다
};
```

> `sort`, `reverse`, `unique`, `join`, `min`, `max`, `index_of`, `count` 등 리스트 조작은 `lst.*`를 사용합니다. [[lst.* 복사 반환 함수](#lst-복사-반환-함수)] 참고.
